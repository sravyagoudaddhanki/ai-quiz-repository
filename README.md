
# AI Quiz Generator

## Project Overview
The AI Quiz Generator is a simple Python-based application 
that asks quiz questions, accepts user answers, checks 
correctness, calculates scores, and displays results 
automatically. It behaves like a basic AI assistant by 
interacting with users through the terminal.

## Project Objective
Develop a Python quiz application that can:
- Ask questions
- Accept answers
- Validate answers
- Calculate score
- Display final result

## Project Description
The application works in the terminal/command prompt and 
allows users to answer quiz questions one by one. The system 
checks answers, updates scores, and displays the final 
result at the end.

## Mandatory Features
- Welcome Message
- Minimum 5 Quiz Questions
- User Answer Input
- Answer Validation
- Score Calculation
- Loop for Multiple Questions
- Final Result Display
- Save Score in Text File
