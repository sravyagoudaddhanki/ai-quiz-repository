# 🤖 AI Quiz Generator

## Project Description
AI Quiz Generator is a Python-based terminal application that automatically generates quiz questions on Python and Artificial Intelligence topics. It checks user answers, calculates scores, tracks time, and displays detailed results.

## Features
- 10 questions database (Python + AI topics)
- Randomly selects 5 questions per session
- Real-time answer validation with explanations
- Score calculation and performance feedback
- Time tracking
- Play again option

## Technologies Used
- **Language**: Python 3
- **Libraries**: random (built-in), time (built-in)
- **Concepts**: Functions, Lists, Dictionaries, Loops, Conditionals, User Input

## How to Run
1. Make sure Python 3 is installed on your computer
2. Open terminal / command prompt
3. Navigate to the project folder
4. Run: python quiz_generator.py

## How It Works
1. User enters their name
2. Program randomly picks 5 questions from the database
3. User answers each question (A/B/C/D)
4. Program checks answer and shows explanation
5. Final score, accuracy % and time are displayed

## AI Workflow Explained
This project demonstrates basic AI workflow:
- Data: Questions stored in structured dictionary format
- Processing: Random selection simulates AI content delivery
- Feedback Loop: Explanations simulate AI tutoring behavior
- Evaluation: Scoring system mirrors AI assessment models
